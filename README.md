# Text-Representation
NLP techniques apply "machines" to process natural language used by human.


 - # Word representation
   - **One-hot encoding**
   - **Bag-of-Words (BOW)**
   - **Term Frequency-Inverse Document Frequency (TF-IDF)**
   - **Word Embedding**
     - Word2Vec
     - GloVe
     - FastText
   - **Transformers-based embedding**

 - # Document representation
   - **Bag-of-Words (BOW)**
   - **Term Frequency-Inverse Document Frequency (TF-IDF)**
   - **Topic modeling**
   - **Document embedding**
     - Word embedding aggregation
     - Doc2Vec
   - **Transformers-based embedding**
